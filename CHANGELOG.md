# Changelog

All notable changes to Dream Journal will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-15

### Added
- Initial release of Dream Journal Emotion Tracker
- Dream entry management (Create, Read, Update, Delete)
- Advanced NLP analysis engine
  - Sentiment analysis using VADER and TextBlob
  - 8-emotion detection system
  - Named entity extraction with spaCy
  - Dream theme identification
  - Stress level detection
- Interactive dashboard with multiple visualizations
  - Emotion timeline charts
  - Emotion distribution pie charts
  - Calendar heatmap
  - Stress trend analysis
  - Theme frequency bar charts
- Insights and predictions
  - Weekly and monthly summaries
  - Pattern detection
  - Mood trend predictions
  - Personalized recommendations
- PDF export functionality
- Tag-based filtering and search
- Sleep quality tracking
- Responsive Bootstrap UI
- Privacy-focused local processing
- SQLite database for data persistence

### Technical
- Flask 3.0.0 backend
- spaCy 3.8+ for NLP
- Plotly for visualizations
- Bootstrap 5 frontend
- RESTful API design
